# Go Projects

5 Go projects built while learning the language, covering CLI tools, REST APIs, authentication, and database integration. Chosen for Go's performance, simplicity, and suitability for backend development.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 01 | Investment Calc | CLI tool that calculates investment returns over time |
| 02 | Banking | Simple banking system with account management and transactions |
| 03 | Notes Taking | REST API for creating and managing notes, backed by SQLite |
| 04 | Price Calc | Pricing calculator with discount and tax logic |
| 05 | Event Booking | REST API for event booking with JWT authentication |

## Technologies
- **Language**: Go
- **Framework**: Gin
- **Database**: SQLite
- **Auth**: JWT (golang-jwt)
