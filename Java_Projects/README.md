# Java Projects

Object-oriented design-pattern implementations written in Java for a graduate-level Object-Oriented Design course. Each demonstrates practical application of Gang of Four patterns.

> **Attribution:** These were completed as coursework, in part as team work with **Robert Diepenbrock** — author credit is shared in the file headers. The State Design Pattern project is predominantly Robert's implementation, kept here as a reference example of the pattern.

## Projects

| Project | Patterns | Description |
|---------|----------|-------------|
| [Factory Design Pattern](Factory_Design_Pattern/) | Factory Method + Strategy | Video-rental pricing system — factories build pluggable pricing, renter-points, and coupon strategies |
| [State Design Pattern](State_Design_Pattern/) | State (+ Factory) | Keystroke-driven calculator that changes behavior as it moves between entry and operator states |
| [Design Pattern Final](Design_Pattern_Final/) | Factory Method + Strategy | Extended rental system with multiple item types (movies, games, books), plus pricing / points / coupon strategies and XML invoice output |

## Technologies
- **Language**: Java
- **Concepts**: OOP, SOLID principles, Gang of Four design patterns (Factory Method, Strategy, State)
