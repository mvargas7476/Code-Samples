# Flutter Projects

7 cross-platform mobile apps built while learning Flutter and Dart, progressing from basic UI components to full-stack apps with Firebase and device-feature integration. Projects are ordered by complexity.

## Projects

| # | App | Description |
|---|-----|-------------|
| 01 | [Roll App](01_roll_app/) | Simple dice roller — intro to Flutter widgets and state |
| 02 | [Quiz App](02_quiz_app/) | Multiple-choice quiz with a results/summary screen and score tracking |
| 03 | [Expense Tracker](03_expense_tracker_app/) | Tracks and categorizes expenses with a bar chart and responsive/landscape layout |
| 04 | [Meals App](04_meals_app/) | Browse, filter, and favorite meals — tab navigation and Riverpod state management |
| 05 | [Shopping List](05_shopping_list_app/) | CRUD shopping list backed by Firebase Realtime Database over HTTP |
| 06 | [Favorite Places](06_favorite_places_app/) | Save places with a camera photo and GPS location on Google Maps, persisted to on-device SQLite |
| 07 | [Chat App](07_chat_app/) | Real-time chat using Firebase Auth and Cloud Firestore |

## Technologies
- **Language**: Dart
- **Framework**: Flutter
- **State management**: Riverpod
- **Backend**: Firebase (Auth, Firestore, Realtime Database)
- **Device features**: camera, geolocation, Google Maps, on-device SQLite (sqflite)
- **Key packages**: flutter_riverpod, google_maps_flutter, image_picker, sqflite, http

> **Note:** The Firebase apps (05, 07) and the maps in Favorite Places (06) require your own API keys / config to run — see each app's README.
