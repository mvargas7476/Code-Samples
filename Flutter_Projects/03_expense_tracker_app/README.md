# Expense Tracker

Add, categorize, and remove expenses, with a bar chart that summarizes spending by category. Includes a modal entry form with a date picker, swipe-to-dismiss list items, and a layout that adapts to landscape.

## What it demonstrates
- App-wide state management and list mutation
- Chart building from aggregated data
- `Dismissible` list items with undo
- Responsive/adaptive layout and theming
- Date formatting and IDs (`intl`, `uuid`)

## Tech
Flutter · Dart · intl · uuid

## Run
```bash
flutter pub get
flutter run
```
