# Meals App

Browse meal categories, view recipe details, filter meals by dietary preference, and mark favorites. Uses tab navigation, a side drawer, and a filters screen, with favorites and filters managed through app-wide providers.

## What it demonstrates
- **Riverpod** state management (meals, favorites, and filters providers)
- Tabbed navigation, a drawer, and a dedicated filters screen
- Hero transitions and image fade-in animations
- Model/data separation (`meal`, `category`, `dummy_data`)

## Tech
Flutter · Dart · flutter_riverpod · google_fonts · transparent_image

## Run
```bash
flutter pub get
flutter run
```
