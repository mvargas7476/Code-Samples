# Robotics Site

A static, multi-page informational website for a junior-high Robotics Magnet program — a project from my earlier career as a Robotics and Programming teacher. It's an early hand-written site (plain HTML/CSS/JS, no framework), included here as a snapshot of where I started before moving into software full-time.

## Pages
- `RoboticsMain.html` — landing page
- `AboutMe.html` — program/instructor intro
- `Curriculum.html` — course overview
- `ContactInfo.html` — contact details

## Tech
HTML5 · CSS3 · vanilla JavaScript (`RoboticsSite.css`, `RoboticsSite.js`)

## View
Open `RoboticsMain.html` directly in a browser — there is no build step.
